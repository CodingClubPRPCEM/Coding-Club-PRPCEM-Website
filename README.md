## 🎨 Design System

### 🌈 Color Palette
```css
--primary-color: #00ff88      /* Matrix Green */
```

### 🔤 Typography

- **Font Sizes**: Responsive clamp() functions
- **Line Heights**: Optimized for readability

## 🛠️ Installation & Setup

### Quick Start
```bash
# Clone the repository
git clone https://github.com/CodingClubPRPCEM/Coding-Club-PRPCEM-Website.git

# Navigate to project directory
cd Coding-Club-PRPCEM-Website

# Open in your preferred code editor
code .

# Serve locally (using any local server)
python -m http.server 8000
# or
npx serve .
# or
live-server
```

### Development Setup
1. **Code Editor**: VS Code recommended with extensions:
   - Live Server
   - CSS Peek
   - HTML CSS Support
   - Prettier

2. **Browser DevTools**: Enable developer mode for testing
3. **Local Server**: Required for full functionality (CORS policies)
