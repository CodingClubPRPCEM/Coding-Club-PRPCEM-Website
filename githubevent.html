About the Event

A hands-on workshop designed to help students master Git, GitHub, and collaboration techniques, along with portfolio deployment.

Why Attend?

Learn Git & GitHub fundamentals

Understand the basics of version control and how to use Git and GitHub effectively.

Work with GitHub Copilot

Experience the power of AI-assisted coding with GitHub Copilot.

Gain collaborative coding experience

Learn how to collaborate on projects using GitHub's powerful features.

Build & deploy a personal portfolio website

Create and deploy your own portfolio website to showcase your skills.

Avail the GitHub Student Developer Pack

Get access to a suite of free tools and resources with the GitHub Student Developer Pack.

Receive an ISTE-certified certificate

Earn a certificate from ISTE to add to your professional credentials.
